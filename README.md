# Sa//t 個人網站

這是我的個人網站，展示了我的作品和一些關於我的資訊。如果有任何建議或意見，歡迎告訴我！

## 目錄結構

- `index.html` - 網站的主頁，包含導航列和一些基本資訊面。
- `style.css` - 網站的樣式表，定義了網站的外觀和佈局。
- `script.js` - 網站的 JavaScript 文件，包含互動功能。
- `images/` - 存放網站使用的圖片資源。

## 使用技術

- HTML5
- CSS3
- JavaScript
- Font Awesome - 用於圖案顯示

## 如何運行

1. 複製此連結至電腦終端機執行：

   ```bash
   git clone https://github.com/your-username/Sa1-t.github.io.git
   ```

2. 打開 `index.html` 文件即可在瀏覽器中查看網站。
